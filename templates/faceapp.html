Face Recognition Application

Face Recognition and Gender Classification: This app is driven by data and machine learning model. When an image is uploaded to the app, it convert the image into grayscale and crops the face from the image using Haar Cascade Classifier. It then converts it to eigen image and finally uses Support Vector Machines to predict the gender of the image. All of this functionality is rendered using a flask app.
